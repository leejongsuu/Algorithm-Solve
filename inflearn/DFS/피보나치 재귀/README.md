### 분류

DFS

### 문제 설명

<p>
1) 피보나치 수열을 출력한다.

2) 입력은 피보나치 수열의 총 항의 수이다.

만약 7이 입력되면 1 1 2 3 5 8 13을 출력하면 된다.
</p>


### 입력

 <p>첫 줄에 총 항수 N(3≤N≤45)이 입력된다.</p>

### 출력

 <p>첫 줄에 피보나치 수열을 출력한다.</p>

### 풀이 

<p>
예를 들어 n = 5 라고 가정해보자.

1 1 2 3 5 

DFS(5) = DFS(3) + DFS(4)

DFS(4) = DFS(2) + DFS(3)

DFS(3) = DFS(1) + DFS(2)

DFS(2) = 1

DFS(1) = 1

ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ

DFS(1) = 1

DFS(2) = 1

DFS(3) = 1 + 1 = 2

DFS(4) = 1 + 2 = 3

DFS(5) = 2 + 3 = 5 이다.

—> 1 1 2 3 5 이런식으로 출력되려면

for문을 돌려서 DFS(1) ~DFS(5)를 출력하면 된다.
</p>
